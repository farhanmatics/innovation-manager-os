# Case Studies

This directory stores analyzed projects as reusable product intelligence.

## Purpose

Every project analyzed through the Innovation OS framework should be stored here.
This creates a **product memory** that future AI agents can reference.

## File Naming Convention

`[project-name].md` — Use the project's code name or a descriptive slug.
